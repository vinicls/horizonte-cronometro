\# 🥋 Cronômetro Horizonte Jiu-Jitsu (v1.8)



Um cronômetro progressivo e inteligente desenvolvido especificamente para o gerenciamento de treinos de Jiu-Jitsu. Este aplicativo web foi projetado para ser executado em qualquer dispositivo, com foco especial em \*\*Smart TVs e dispositivos móveis\*\* dentro do tatame.





\## 🎯 Finalidade



O objetivo deste projeto é oferecer uma interface visual clara, sonora e automatizada para instrutores e atletas. Ele resolve o problema comum de visibilidade em ambientes grandes (academias), adaptando sua interface dinamicamente quando o cronômetro está em execução para maximizar a leitura do tempo.



\## 🚀 Recursos Principais



\### 1. Interface Adaptativa (TV \& Mobile)

\- \*\*Modo TV:\*\* Ao iniciar o cronômetro, os controles de configuração desaparecem suavemente, permitindo que o tempo e os rounds ocupem 100% da tela.

\- \*\*Responsividade Total:\*\* Layouts otimizados para smartphones (controles em sanfona) e desktops/tablets.

\- \*\*Dark Mode Nativo:\*\* Alternância entre temas claro e escuro para melhor contraste dependendo da iluminação do dojo.



\### 2. Gestão de Rounds e Presets

\- \*\*Presets Rápidos:\*\* Configurações pré-definidas para categorias Kids, Juvenil, Adulto, Master e Treino Livre.

\- \*\*Customização Total:\*\* Ajuste fino de tempo de luta, tempo de descanso (de 10s a 3min), número de rounds e alertas finais.

\- \*\*Indicadores Visuais (Dots):\*\* Sistema de LEDs virtuais que indicam rounds concluídos, round atual e período de prontidão.



\### 3. Áudio Sintetizado (Web Audio API)

Diferente de outros cronômetros, este app \*\*não utiliza arquivos MP3 externos\*\*. Os sons são gerados em tempo real pelo navegador:

\- \*\*Apito de Início:\*\* Som de alta frequência senoidal.

\- \*\*Fim de Round:\*\* Som grave e potente (onda quadrada/sawtooth).

\- \*\*Beeps de Contagem:\*\* Alertas curtos nos últimos segundos de descanso ou aviso final.



\### 4. Funcionalidades Avançadas

\- \*\*Aviso de Prontidão:\*\* Alerta sonoro e visual 10 segundos antes do fim do intervalo.

\- \*\*Barra de Progresso:\*\* Feedback visual rápido da duração do round.

\- \*\*Relógio Interno:\*\* Exibição de hora e data no painel de controle.



\## 🛠️ Tecnologias Utilizadas



Este projeto foi construído utilizando o conceito de \*\*Vanilla Web Development\*\*, garantindo leveza e zero dependências externas:



\- \*\*HTML5:\*\* Estrutura semântica e metatags para Web App (PWA ready).

\- \*\*CSS3:\*\* Variáveis nativas, Flexbox, CSS Grid e animações complexas via `@keyframes`.

\- \*\*JavaScript (ES6):\*\* Lógica de estado, manipulação de DOM e síntese de áudio via `AudioContext`.



\## 📦 Como Usar



Por ser um arquivo único (`.html`), a execução é extremamente simples:



1\. Faça o download do arquivo `cronometro-jiujitsu.html`.

2\. Abra-o em qualquer navegador moderno (Chrome, Safari, Edge ou navegador de Smart TV).

3\. (Opcional) No celular, selecione "Adicionar à tela de início" para usá-lo como um aplicativo nativo.
